# Retail Sales Data Analysis

A comprehensive data analysis project demonstrating end-to-end retail analytics using Python and Pandas. This project showcases key skills for a Senior Data Analyst position in the Australian market.

## Project Overview

**Purpose:** Demonstrate retail sales data analysis skills including data cleaning, exploratory analysis, KPI calculation, and visualization.

**Key Features:**
- Data generation and cleaning
- Exploratory Data Analysis (EDA)
- KPI calculations and metrics
- Regional and categorical analysis
- Visualization with matplotlib/seaborn

## Skills Demonstrated

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Data Cleaning & Preprocessing
- Statistical Analysis
- Business Intelligence
- Data Visualization
- KPI Design & Reporting

## Getting Started

### Prerequisites

- Python 3.8+
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/jackyang199/retail-sales-analytics.git
cd retail-sales-analytics

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Running the Analysis

```bash
python analysis.py
```

## Analysis Steps

1. **Data Generation** - Create realistic retail sales data
2. **Data Cleaning** - Handle missing values, format dates
3. **EDA** - Explore distributions and patterns
4. **KPI Calculation** - Compute key metrics
5. **Regional Analysis** - Analyze by Australian cities
6. **Product Analysis** - Category and product performance
7. **Visualization** - Generate charts for dashboard

## Key Metrics Analyzed

| Metric | Description |
|--------|-------------|
| Total Sales | Overall revenue |
| Average Order Value | Mean transaction size |
| Customer Segments | Premium/Regular/New/VIP |
| Top Categories | Best-performing product types |
| Regional Performance | Sales by Australian cities |
